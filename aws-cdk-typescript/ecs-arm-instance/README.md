# Demo repo for ecs Task Definition  use arm64 container via use EC2 Arm instance.


## To Install
```bash
yarn
```

## To Diff
```bash
npx cdk diff
```

## To synth
```bash
npx cdk synth
```

## To Deploy
```bash
npx cdk deploy
```

## To Hotswap Deploy ( will save your a lot of time!!! )
```bash
npx cdk deploy --hostswap
```
